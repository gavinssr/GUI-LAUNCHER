# Artifact Protocol

## 每次任务最少输出

- changed files
- package scope
- promotion candidate
- 风险/阻塞
- 下一步（P0/P1）

## 推荐附带内容

- 关键命令结果摘要
- 验证截图或访问结果摘要
- build / lint / typecheck / registry 检查摘要

## Cloud Agent PR Expectations

- 说明 target package
- 说明 task layer
- 说明 verification steps
- 说明 artifacts 与 residual risk
