# Promotion Reviews

本目录存放 promotion candidate 的正式复盘与裁决记录。

## 何时写入

- 某次 consumer / semantic / registry 交付出现 promotion candidate
- 某轮 doc-gardening 需要做统一“是否上推”复盘
- 某个实验型样本需要补正式不上推结论

## 最小字段

- candidate scope
- evidence
- decision
- rationale
- owner
- next review trigger

## 周期建议

- 每完成一个大 phase，至少复盘一次当前 promotion candidate
- 如无候选项，也记录一次明确的 `No Promotion` 基线
